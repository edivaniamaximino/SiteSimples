<h1>LogFull - Website </h1>

Este projeto apresenta um site institucional fictício da LogFull, desenvolvido com HTML5, CSS3 e JavaScript, com foco em layout moderno, experiência do usuário e responsividade, simulando a presença digital de uma empresa de soluções logísticas inteligentes.

O site conta com navegação fluida, menu responsivo em formato hambúrguer para dispositivos móveis, animações suaves e uma identidade visual inspirada em interfaces modernas e minimalistas.<h2>Objetivo do Projeto</h2>

Criar um site institucional simples, elegante e funcional que possa ser utilizado como base para projetos de logística, transporte, supply chain ou empresas que desejam apresentar seus serviços de forma clara e profissional.
<h2>Tecnologias Utilizadas</h2>

- **HTML5** — Estrutura semântica e organizada
- **CSS3** — Estilização moderna, layout responsivo e design inspirado em interfaces premium
- **JavaScript** — Manipulação de classes e eventos
- **Google Fonts** — Tipografia moderna (Inter)
- **Design Responsivo** — Adaptado para desktop, tablet e mobile

<h2>Funcionalidades do Projeto</h2>

Página inicial com **hero section** e chamada institucional

Página de **Soluções** com apresentação clara dos serviços logísticos 

Página de **Contato** com formulário elegante e focado em conversão 

Header fixo com navegação intuitiva  

Design dark moderno inspirado em marcas tech  

Estrutura reutilizável para futuras expansões
